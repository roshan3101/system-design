# Creational Patterns

Patterns that manage object creation:
- Builder (fluent, director, and step variations for HTTP requests)
- Factory (simple, factory method, abstract factory)
- Prototype (clone character loadouts)
- Singleton (single instance access)

Each subfolder has its own `README.md` and `main.cpp`. Build with `g++ -std=c++17 -Wall -Wextra -o demo main.cpp`.
